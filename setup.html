Getting Started with Putt Window: External Camera Putting Setup

Welcome to Putt Window, your ultimate mixed reality putting experience! Follow these steps to use your real ball and putter with Putt Window using an external camera setup.

What You'll Need

  • A mixed reality headset and a copy of Putt Window.
  • A real golf ball and putter.
  • An external camera-based putting setup (either Exputt or a webcam).
  • A laptop or desktop PC running Windows 10 or newer, and the MLM2PRO-GSPro Connector software.

Step 1: Download the MLM2PRO-GSPro Connector

1. Visit the MLM2PRO-GSPro Connector GitHub repository.

2. Follow the instructions provided in the repository to download and install the MLM2PRO-GSPro Connector on your PC.

Step 2: Set Up Your External Camera Based Putting System

Option A: Exputt Putting Setup

If you're using an Exputt:

  • Follow the instructions in the repository readme on how to set up the Exputt with the connector.

Option B: Webcam Putting Setup

If you're using a webcam:

  • Follow the instructions in the repository readme on how to set up the webcam with the connector.

Step 3: Configure the MLM2PRO-GSPro Connector

Putt Window is compatible with the OpenAPI v1, so the connector software will connect to it as if it were connecting to GSPro.

1. Once your camera-based setup is ready, open the MLM2PRO-GSPro Connector on your PC.

2. In the connector settings, configure the putting settings to work with your camera setup (either Exputt or the webcam).

3. Set the destination IP address to the IP address of your mixed reality headset.

4. Ensure that the port number is set to 1921, which is used by Putt Window instead of the standard GSPro port 0921.

Step 4: Connect the MLM2PRO-GSPro Connector to Putt Window

1. Ensure your mixed reality headset is connected to the same network as the PC running the MLM2PRO-GSPro Connector.

2. Launch Putt Window in your mixed reality headset.

3. In the MLM2PRO-GSPro Connector software, point it to the headset IP address, as if connecting to GSPro.

4. Start the connection, and the connector will now relay the shot data to Putt Window.

Step 5: Adjust Alignment in Putt Window

1. Enter setup mode in Putt Window.

2. Follow the instructions in the headset to align the origin and hitting direction of the virtual putting green with your real putting setup.
